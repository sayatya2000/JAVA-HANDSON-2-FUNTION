output is Undefined

5.Print output: 

                var x = 21;
                girl ();
                console.log(x)
                function girl() {
                    console.log(x);
                    var x = 20;
                };

Ans.

First output is Undefined and Second output is 21.

This code defines a variable x with the value 21, then calls a function expression girl that logs the value of x to the console and then declares and initializes a local variable x with the value 20. However, since the girl function is called before the local variable x is declared, the output of the first console.log(x) statement inside the function is undefined. After the girl function call, the output of the final console.log(x) statement is 21 because it references the global variable x defined outside the function scope.
